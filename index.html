Estructura Básica de HTML

Un documento HTML comienza con <!DOCTYPE html> seguido de las etiquetas <html>, <head>, y <body>.

Etiquetas Semánticas

Las etiquetas semánticas como <header>, <nav>, <main>, <section>, <article>, <aside>, y <footer> ayudan a estructurar el contenido de manera lógica.

Negrita y Cursiva

Para poner texto en negrita se usa <strong> y para cursiva se usa <em>. Ejemplo: <strong>negrita</strong> y <em>cursiva</em>.

Transparencia en el Fondo

Para darle transparencia al fondo de un elemento se usa rgba o hsla en CSS. Ejemplo: background-color: rgba(0, 0, 0, 0.5);
